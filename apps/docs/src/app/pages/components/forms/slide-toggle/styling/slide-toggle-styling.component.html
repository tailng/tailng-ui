Styling

tng-slide-toggle exposes rootKlass, labelKlass, inputKlass; base trackKlass / thumbKlass (both states); and per-state trackOnKlass, trackOffKlass, thumbOnKlass, thumbOffKlass.

rootKlass

Applies to the root wrapper. Default: inline-flex items-center gap-2 select-none.

Example

trackKlass

Applies to the track (pill background). Use for color or size.

Example

thumbKlass

Applies to the thumb (moving circle). Use for size or shadow.

Example

trackOnKlass / trackOffKlass, thumbOnKlass / thumbOffKlass

Per-state overrides when toggle is on vs off. Merged with base track/thumb and component defaults.

Example

labelKlass

Applies to the label <span>. Default: text-sm text-fg.
