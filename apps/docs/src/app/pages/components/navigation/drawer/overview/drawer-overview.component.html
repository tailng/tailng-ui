Overview

tng-drawer is a slide-out panel (left, right, top, bottom). Control visibility with [open]; handle (closed) to set open to false. Closes on backdrop click and Escape by default. Supports focus trap. Use placement and class hooks (backdropKlass, panelKlass, sizeKlass, heightKlass).
