API reference for dialog

Inputs

Controlled open and behavior options.

Outputs

Klass hooks

Directive: tngDialogInitialFocus

Place on the element that should receive focus when the dialog opens (e.g. primary action button). Import from @tociva/tailng-ui/overlay with TngDialog.

Content projection

Default slot = body. [tngDialogHeader] = header section, [tngDialogFooter] = footer section.
