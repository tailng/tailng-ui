Overview

tng-table renders tabular data from a rows array. Define columns with tng-col: id, header, [value] (row accessor), optional width and align. Use ng-template tngCell and ng-template tngHeader for custom cell/header content. Supports client-side sorting and class hooks for styling.

Features

Columns

tng-col: id, header, value (function or property key), width, align (left | right | center), klass. Optional tngCell / tngHeader templates.

Sorting

Client-side sort via sortMode="client" and tng-sort-header; (sortChange) for server or client. rowKey for stable row tracking.

Styling

tableKlass, theadKlass, thKlass, tdKlass, tbodyKlass. emptyText when rows are empty.
