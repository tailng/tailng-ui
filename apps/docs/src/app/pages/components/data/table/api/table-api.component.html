TngCol inputs

Cell and header templates

Inside tng-col, use ng-template tngCell with let-row, let-value="value" (and optional let-rowIndex). Use ng-template tngHeader with let-header="header" for custom header content. Import TngCellDef and TngHeaderDef from @tociva/tailng-ui/table.
